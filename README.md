
# Future Interns – Prompt Engineering Task 2


AI Content Marketing Using UGC Ads
Project Overview
This project demonstrates how Prompt Engineering and Artificial Intelligence can be used to generate high-converting User Generated Content (UGC) advertisements for beauty and skincare brands.
Modern consumers respond more positively to authentic content that feels like genuine customer experiences rather than traditional advertisements. This project focuses on creating UGC-style marketing assets for a real skincare product using structured AI prompts.
The generated content is designed for Instagram Reels, TikTok, YouTube Shorts, paid social media advertisements, and influencer marketing campaigns.

Reusable Prompt System

The prompt framework was designed to be reusable across multiple beauty and skincare brands.

Only the following inputs need to be changed:

- Product Name
- Product Benefits
- Target Audience
- Brand Positioning

The same prompts can generate:

- Hooks
- UGC Scripts
- CTAs
- Captions
- Hashtags
- Platform-Specific Content
- Video Storyboards

for any skincare, beauty, wellness, or personal care brand.

This makes the system scalable and useful for agencies managing multiple clients.
